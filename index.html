🗂 기수 · 과 이동
  • 기수 선택 — 상단 왼쪽 드롭다운에서 연도·기수(2026년 1기/2기…) 선택
  • 과 선택 — 상단 오른쪽 드롭다운에서 과 선택, 또는 화면을 좌우 스와이프
  • 이전과 / 다음과 — 하단 ◀ 이전 과 · 다음 과 ▶ 버튼으로 이동
  • 즐겨찾기 ☆ — 카드 왼쪽 상단 ☆ 탭으로 즐겨찾기 등록·해제. 설정 › 즐겨찾기 탭에서 목록 확인.
🧠 5단계 암송
  • 하단 ‹ › 버튼 또는 점(●)으로 단계 전환. 현재 단계는 왼쪽 하단에 표시.
  • 1단계 전체보기 — 기억절 본문 전체를 확인하며 시작
  • 2단계 부분숨김 — 일부 단어가 회색 박스로 가려짐
    • 가려진 단어 탭 → 즉시 흐리게 보임 → 다시 탭 → 또렷하게 → 다시 탭 → 다시 숨김
    • 중앙 ↺ 버튼 = 숨길 단어를 새로 무작위로 섞기
  • 3단계 한글자만 — 한국어는 마지막 글자(어미/조사), 그 외 언어는 첫 글자만 표시
  • 4단계 전체숨김 — 모든 단어가 가려짐
    • 단어 탭 → 즉시 흐리게 → 다시 탭 → 또렷하게 → 다시 탭 → 다시 숨김
  • 5단계 그림연상 — 과 이미지를 보며 기억절 내용을 떠올림
    • 기본 이미지는 자동 표시, 이전과·다음과 사이 🖼 버튼으로 나만의 이미지 등록 가능
    • 이미지 등록: 파일 선택 · 드래그 앤 드롭 · 클립보드 붙여넣기(Ctrl+V) · 4MB 이하 자동 압축
  • 모두보기 (오른쪽 상단 단계 줄) — 현재 단계에서 모든 단어를 한꺼번에 보임. 다시 탭하면 원래대로.
✓ 암송 체크
  • 카드 오른쪽 상단 ✓ 버튼으로 암송 완료 횟수 기록
  • 짧게 탭 = 횟수 +1  |  길게 누름(0.6초↑) = 횟수 −1 (실수 수정)
  • 1회 이상 = 완료(초록), 5회 이상 = 숙달(금색)로 표시되며 통계에 반영
🖍 형광펜
  • 성경절 구절 오른쪽 ✦(펜 아이콘) 버튼으로 형광펜 모드 활성화
  • 세로 팔레트에서 색 선택 후 단어를 탭하면 색이 적용됨
  • 같은 색 단어끼리 하나의 구(句)로 묶여 암송 단위 구분에 활용
  • 팔레트 하단 전체 삭제로 현재 과 하이라이트 모두 제거
🔊 음성
  • 상단 🔊 버튼으로 음성 패널 열기
  • 기본 음성 — 개발자 제공 MP3 재생 (한국어 2기 기준)
  • 내 목소리 녹음 — 언어 탭(한/EN/日/中/IN) 선택 후 직접 녹음·재생·삭제
    • 언어별로 독립 저장되며, 녹음 있는 탭에 ● 표시
⚙️ 표시 설정
  • 글자 크기 — 카드 하단 가− · 가+ 버튼으로 조절
  • 폰트 — 가+ 오른쪽 T 버튼으로 폰트 변경 (시스템·명조·돋움·고딕)
  • 언어 🌐 — T 오른쪽 🌐 버튼으로 주 언어·보조 언어 설정 (최대 3개 병렬 비교)
🔍 검색
  • 상단 🔍 버튼으로 전체 기억절 검색 (내용·장절·과 제목)
  • 검색 결과 탭하면 해당 기수·과로 바로 이동
📦 모듈
  • 모듈은 기억절 데이터 묶음입니다. 기수별 기억절·주제장 암송·특별 묵상 세트 등을 독립적으로 설치·제거할 수 있습니다.
  • 설치된 모듈 — 설정 › 모듈 탭에서 확인. 드롭다운에 자동으로 나타납니다.
  • 모듈 추가 — .module.json 파일을 가져오거나, 📦 모듈 빌더로 직접 제작 후 설치합니다.
  • 모듈 이름 수정 — 모듈 카드의 ✏ 버튼으로 한글 이름으로 변경 가능. 모듈 ID(파일·DB 내부 키)는 변경되지 않습니다.
  • 순서 조정 — 모듈 카드의 ▲▼ 버튼으로 드롭다운 목록 순서를 바꿀 수 있습니다.
  • 모듈 내보내기 — ↓ 버튼으로 .module.json 파일 저장. 다른 기기나 다른 사람과 공유 가능.
  • 모듈 제거 / 재설치 — 제거 시 구절 데이터만 삭제되고 형광펜·암송 기록은 보존됩니다. 재설치하면 즉시 복구됩니다.
  • 모듈 빌더 — 별도 도구(module-builder.html)로 나만의 기억절 모듈을 4단계로 제작합니다.
    • 1단계: 모듈 ID(영문), 제목(5개 언어), 테마 색상 설정
    • 2단계: 언어별 구절 텍스트 붙여넣기 → 자동 파싱 (Bethlehem·파이프·블록 포맷 지원)
    • 3단계: 음성(MP3)·이미지 파일 첨부
    • 4단계: 미리보기 확인 후 .module.json 다운로드 → 앱에서 설치
⚙ 설정 패널 탭 안내
  • 내정보 — 사용자 이름·가장 좋아하는 성경절 입력. 앱 시작 화면에 표시됨.
  • 즐겨찾기 — 즐겨찾기한 과 목록 확인 및 이동
  • 내성경절 — 나만의 기억절 추가·편집·삭제. 다국어(5개 언어) 직접 입력 지원. 음성 파일 첨부 가능.
  • 통계 — 기수별 암송 현황 (완료/숙달) 확인
  • 모듈 — 기억절 데이터 묶음 관리. 설치·제거·재설치·이름 수정·순서 변경·내보내기.
  • 백업 — 전체 데이터(사용자 성경절·형광펜·암송기록·녹음·이미지·프로필·모듈)를 ZIP으로 내보내기·가져오기. 기기 간 이전, 앱 초기화 전 백업에 활용.
